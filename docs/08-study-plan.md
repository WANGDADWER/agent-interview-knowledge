# 六周快速入门计划

| 阶段 | 目标 | 学习内容 | 产出物 |
| --- | --- | --- | --- |
| 第 1 周 | 建立 Agent 基础概念 | Agent loop、tool calling、memory、planning、RAG 基础 | 一个会调用 2 个工具的最小 Agent |
| 第 2 周 | 掌握 RAG 工程 | LlamaIndex 数据接入、chunk、embedding、retrieval、rerank、citation | 本地知识库问答 Demo + 20 条评测集 |
| 第 3 周 | 掌握 LangChain/LangGraph | StateGraph、节点/边/状态、checkpoint、HITL、LangSmith tracing | 一个有状态、可恢复的业务流程 Agent |
| 第 4 周 | 理解多 Agent | AutoGen/MAF 思想、supervisor-worker、group chat、角色分工、冲突解决 | 一个多 Agent 调研或客服系统 |
| 第 5 周 | 工程化与安全 | 权限、审计、prompt injection、防重复执行、成本优化、监控指标 | 加入权限控制、日志、错误恢复和评测报表 |
| 第 6 周 | 面试表达与项目包装 | 系统设计题、项目 STAR、性能指标、失败案例复盘 | 2 分钟项目介绍 + 10 个追问答案 |
